# club

## Features
- Core club management features for the korfbal project.

## Requirements
- Django >= 3.2

## Usage
Add to `INSTALLED_APPS` in your Django settings:
```python
INSTALLED_APPS = [
    ...
    'club',
]
```

## Testing
To run tests for this app:
```bash
python manage.py test club
```
Make sure you have a test database configured.
