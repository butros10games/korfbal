# player (Django app)

## Features

- Player management and profiles for the korfbal project.

## Requirements

- Django >= 3.2

## Usage

Add to `INSTALLED_APPS` in your Django settings:

```python
INSTALLED_APPS = [
    ...
    'player',
]
```

## Local test

Run from the project root:

- uv run python manage.py test player
