# common

## Features

- Shared utilities and helpers for the korfbal project.

## Requirements

- Django >= 3.2

## Usage

Add to `INSTALLED_APPS` in your Django settings:

```python
INSTALLED_APPS = [
    ...
    'common',
]
```

## Testing

To run tests for this app:

```bash
python manage.py test common
```

Make sure you have a test database configured.
